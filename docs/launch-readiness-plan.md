# Launch Readiness Plan

1. Real paid checkout test
   - Create a fresh account, buy Creator or Pro, confirm the plan and credits update, open billing portal, cancel, and confirm the account returns to Free.

2. End-to-end customer flow test
   - Sign up, analyze a niche, confirm idea-card images, save an idea, add one to calendar, generate hooks/hashtags/script, subscribe to alerts, and test on mobile.

3. Cost protection test
   - Confirm logged-out users cannot generate, free users are blocked after credits run out, paid users decrement credits correctly, and failed analyses do not charge credits.

4. Support setup
   - Decide the support email, refund language, bad-output process, and whether dashboard feedback should send to email, database, or both.

5. Analytics and error tracking
   - Track signups, checkout starts/completions, analysis failures, AI/image costs, popular niches, and funnel drop-off.

6. Final marketing polish
   - Tighten homepage examples, screenshots, credit explanation, pricing copy, FAQ, and platform-scanning claims.

7. Small beta launch
   - Let 5-10 real users try it, fix confusion or bugs, then start broader marketing.
